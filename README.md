## Bookstore SQL Analysis

**Tools Used:** PostgreSQL, SQL

**About the Project:**
Analysed a bookstore database with 3 tables — Books, Customers, and Orders — 
to extract business insights using SQL queries.

**Key Insights:**
- Total revenue generated from all orders
- Top selling genres and authors
- Most frequently ordered books
- Customer spending patterns
- Remaining stock after order fulfillment

**Skills Demonstrated:**
SELECT, WHERE, JOIN, GROUP BY, HAVING, AGGREGATE FUNCTIONS, SUBQUERIES
